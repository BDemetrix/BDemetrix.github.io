##### Окружение:
1)  ```  npm i  ```
2)  ```  npm install webp-converter@2.2.3 --save-dev  ```
3) запуск  ```  gulp  ```

Переустановка node-sass:   ```  npm rebuild node-sass  ```

Проблема с pyton:  ```  npm install --global windows-build-tools  ```
