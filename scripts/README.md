# scripts/

## generate-pdf.js

Генерирует 4 PDF-версии резюме из десктопного вида страницы (1440px).

### Результат

```
pdf/
  resume-ru-dark.pdf
  resume-ru-light.pdf
  resume-en-dark.pdf
  resume-en-light.pdf
```

Кнопка **Download Resume PDF** на сайте автоматически скачивает нужный файл
в зависимости от текущей темы и языка пользователя.

### Использование

```bash
# Установить зависимости (один раз)
npm install

# Сгенерировать все 4 PDF
npm run pdf
```

### Когда перегенерировать

- После любых изменений в `index.html`, `css/style.css`, `js/i18n.js`
- После обновления фото, проектов, опыта
- После смены шрифтов или цветовых переменных

### Как работает

1. Поднимает локальный HTTP-сервер на `localhost:3099`
2. Открывает Puppeteer (headless Chrome) с viewport `1440×900`, `deviceScaleFactor: 2`
3. Для каждой комбинации `lang × theme` устанавливает `localStorage` и перезагружает страницу
4. Активирует все `._scr-item` (снимает scroll-анимации)
5. Сохраняет PDF на всю высоту страницы без разбивки на страницы

### Требования

- Node.js 18+
- `npm install` (устанавливает `puppeteer`)
